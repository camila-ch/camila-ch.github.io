Sitio web dinámico con base de datos

En este proyecto, debimos crear un sitio web dinámico utilizando el framework web Bottle, la base de datos SQLite y Bootstrap para la visualización gráfica. Para ello se requirió: una página de inicio con una lista de todos los datos y su imagen, una vista detallada de cada dato en una página separada, la posibilidad de añadir un dato y un botón para eliminarlo. Para mi ejemplo, elegí hacer de mi sitio web un catálogo de distintas obras de distintos artistas. Los usuarios pueden elegir entre una lista desplegable de artistas, añadir una imagen del cuadro, el nombre del cuadro y la fuente de la imagen. La información del artista se añade automáticamente al elegirlo.

No hay opción 'en vivo' ya que el código contiene Python y las páginas de Github no son compatibles con sitios web dinámicos, por lo que no se puede desplegar allí.
