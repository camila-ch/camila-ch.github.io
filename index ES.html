En pocas palabras

Nací en Colombia y crecí en Nueva York; actualmente termino mi máster en Comunicación y Tecnologías de la Traducción en la Universidad de Ginebra. Trabajos

Trabajo de curso de multimedia, web y diseño

Esta clase me permitió desarrollar una base sólida en el diseño web, abarcando herramientas y tecnologías esenciales como GitHub para la implementación, Bootstrap para el diseño adaptativo y los fundamentos de SQL, JavaScript, Python y el trabajo con API.

Para el trabajo final, se nos encargó combinar estas habilidades para crear un sitio web dinámico respaldado por una base de datos. Desarrollé una aplicación de catálogo de pinturas utilizando el marco web Bottle y SQLite. El sitio permite a los usuarios seleccionar artistas de un menú desplegable y añadir nuevas entradas, incluyendo el nombre de la pintura, la imagen y la fuente. La información del artista se rellena automáticamente en función de la selección del usuario, lo que agiliza el proceso de introducción de datos.

Debido a las políticas de derechos de autor de la clase, no hay una versión activa del proyecto disponible públicamente; sin embargo, se puede consultar las capturas de pantalla de cada interfaz.
