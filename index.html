Dynamic Website and Databases

In this project, we had to create a dynamic website using the Bottle web framework, the SQLite database and Bootstrap for the display. We were required to include: a homepage with a list of all the entries and their image, a detailed view of each entry on separate page, the ability to add an entry and a button to delete an entry. For my example, I chose to make my website a catalogue of different paintings by different artists. Users can choose from a drop-down list of artists, add an image of the painting, the painting name and image source. The artist's information is automatically populated when an artist is chosen.

No 'live' option as the code contains Python and Github pages does not support dynamic sites, so it cannot be deployed on there.
