Quick Bio

I'm an English-Spanish bilingual and fluent in French. In another life I briefly worked as a French pastry cook.

Multimedia Web & Design Coursework

This course provided a strong foundation in web development, covering essential tools and technologies such as GitHub for deployment, Bootstrap for responsive design, and the fundamentals of SQL, JavaScript, Python, and working with APIs.

For the final project, we were tasked with combining these skills to build a dynamic website backed by a database. I developed a painting catalogue application using the Bottle web framework and SQLite. The site allows users to select artists from a dropdown menu and add new entries, including the painting’s name, image, and source. Artist information is automatically populated based on the user’s selection, streamlining the data entry process.
